# Velog Backup

> Velog 블로그 포스트를 안전하게 백업하는 무료 웹 서비스

## 주요 기능

- **GitHub 로그인**: GitHub 계정으로 간편하게 시작
- **서버 저장**: 포스트가 서버에 안전하게 저장됨
- **영구 보존**: Velog에서 삭제해도 백업은 유지
- **마크다운 다운로드**: 언제든지 .md 파일로 내려받기
- **완전 무료**: 모든 기능 무료 제공

## 사용 방법

### 1. 회원가입
1. [velog-backup.vercel.app](https://velog-backup.vercel.app) 접속
2. "GitHub로 시작하기" 클릭
3. GitHub 계정으로 로그인

### 2. Velog 계정 연동
1. 대시보드에서 Velog 사용자명 입력 (예: `username`)
2. "확인" 버튼 클릭

### 3. 백업 시작
- "지금 백업하기" 버튼으로 즉시 백업
- 모든 포스트가 서버에 저장됨

### 4. 포스트 확인 및 다운로드
1. 대시보드에서 "포스트 보기" 클릭
2. 백업된 포스트 목록 확인
3. 개별 포스트를 마크다운 파일로 다운로드

## 자주 묻는 질문

**Q: 비용이 드나요?**
A: 완전 무료입니다.

**Q: 이미지도 백업되나요?**
A: 현재는 이미지 URL만 저장됩니다. 이미지 파일은 Velog 서버에서 제공됩니다.

**Q: Velog 포스트가 삭제되면 백업도 삭제되나요?**
A: 아니요. 서버에 저장된 백업은 그대로 유지됩니다.

**Q: 다른 사람이 내 백업을 볼 수 있나요?**
A: 아니요. 각 사용자는 자신의 백업만 볼 수 있습니다.

## 보안

- GitHub OAuth 공식 인증 사용
- 비밀번호 저장 없음
- HTTPS 암호화 전송
- 언제든지 데이터 삭제 가능
